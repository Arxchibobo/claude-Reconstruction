## 💡 如何充分利用这些资源

### 1. 新手入门路径

```
Step 1: 阅读 "skill是什么" → 理解基础概念
        ↓
Step 2: 学习 "如何安装skill" → 掌握安装方法
        ↓
Step 3: 浏览 "skill收藏夹" → 发现实用 Skills
        ↓
Step 4: 尝试 PPT 生成工具 → 实战练习
        ↓
Step 5: 加入学习群 → 持续学习和交流
```

### 2. Skills 发现流程

1. **浏览集合网站** - 在多个平台搜索你需要的功能
2. **查看官方仓库** - 检查 Skills 的质量和更新状态
3. **参考群友收藏** - 查看 "skill收藏夹" 中的热门推荐
4. **实践验证** - 安装并测试 Skills 是否满足需求

### 3. 创建自定义 Skills

如果现有 Skills 不满足需求：

1. **提交需求** - [点击此处提交](https://e5zve6mvyq.feishu.cn/share/base/form/shrcnYplqZRoVWgChMQ4djkZFue)
2. **参考即梦工具** - 使用 jimeng-free-api-all 将 Web 应用转为 API
3. **查看官方文档** - 学习如何创建自己的 Skills
