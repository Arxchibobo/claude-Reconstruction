## 🎓 Skills 详解

### 🏆 重点推荐（最常用）

#### 1. `marketing-ideas` - 140 个营销创意库 💎

**用途**：快速获取经过验证的营销策略和战术

**内容分类**：
- 内容与 SEO（10 个）
- 竞品与对比（3 个）
- 免费工具与工程化（9 个）
- 付费广告（13 个）
- 社交媒体与社区（10 个）
- 邮件营销（7 个）
- 合作与计划（10 个）
- 活动与演讲（8 个）
- PR 与媒体（4 个）
- 发布与促销（11 个）
- 产品驱动增长（10 个）
- 内容格式（13 个）
- 非常规与创意（14 个）
- 平台与市场（8 个）
- 国际化与本地化（2 个）
- 开发者与技术（4 个）
- 其他（4 个）

**使用示例**：
```
你："我需要一些营销点子"
Claude：让我为你推荐 3-5 个最适合你当前阶段的创意...

你："有没有低成本的营销方法"
Claude：推荐以下免费/低成本策略：
1. 内容营销 + SEO
2. 社区建设
3. 社交媒体...
```

#### 2. `marketing-psychology` - 70+ 心理学模型 🧠

**用途**：应用行为科学和认知偏差提升营销效果

**内容覆盖**：
- 认知偏差（锚定效应、损失厌恶、社会证明等）
- 心理触发器（稀缺性、紧迫性、互惠等）
- 决策框架（系统 1/2 思维、启发式等）
- 行为模式（习惯形成、承诺一致性等）

**使用示例**：
```
你："如何让用户更快做决策"
Claude：应用以下心理学原理：
1. 默认效应 - 预选推荐方案
2. 选择悖论 - 限制选项数量
3. 社会证明 - 展示他人选择...
```

#### 3. `copywriting` - 专业文案撰写 ✍️

**核心原则**：
- 清晰 > 聪明
- 收益 > 功能
- 具体 > 模糊
- 用户语言 > 公司术语

**页面结构框架**：
- Hero 区域（标题/副标题/CTA）
- 社会证明
- 问题/痛点
- 解决方案/收益
- 工作原理（3-4 步）
- 详细社会证明
- 异议处理
- 最终 CTA

**文案公式**（10+ 种）：
```
1. {达成目标} without {痛点}
   例：Understand users without drowning in numbers

2. Never {不愉快事件} again
   例：Never miss a sales opportunity again

3. {问题} → {解决方案}
   例：Hate returning stuff to Amazon? → 我们的方案
```

#### 4. `page-cro` - 页面转化优化 📈

**优化框架**：
1. 数据驱动分析
2. 用户心理洞察
3. 设计/文案改进
4. 测试验证

**审计清单**：
- 价值主张清晰度
- CTA 可见性和说服力
- 页面加载速度
- 移动端体验
- 社会证明质量
- 表单复杂度
- 异议处理

**使用示例**：
```
你："这个落地页转化率只有 2%，怎么提升"
Claude：
1. 分析当前页面
2. 识别主要问题
3. 提供优化方案
4. 建议测试计划
```
