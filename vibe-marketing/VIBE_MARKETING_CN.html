核心模板详解

市场差距分析

包含：竞争对手盲点、蓝海机会、搜索情报、趋势识别

内容差距分析

包含：主题差距、格式差距、定位差距、问题差距

收入预测

包含：流量假设、转化漏斗、收入流、低/中/高场景

网红模式

包含：语言分析、开场钩子、可重复结构、效果指标

优先内容计划

包含：今日速赢、本周目标、未来发展

实际应用场景

场景1: 竞品分析

步骤：

1. 用Firecrawl抓取竞品网站

2. 提取他们的定位、ICP、UVP

3. 用Perplexity搜索市场趋势

4. 让Claude分析差距机会

产出：竞品分析报告 + 差异化策略

场景2: 内容策划

步骤：

1. 分析目标受众的问题/痛点

2. 用Perplexity找高搜索低竞争话题

3. 参考网红模式设计内容框架

4. 让Claude生成内容草稿

产出：内容日历 + 首篇文章

场景3: 市场定位

步骤：

1. 审计自己的网站定位

2. 对比3-5个竞品

3. 识别未被占领的定位角度

4. 重新定义品牌信息

产出：品牌定位文档 + 信息框架

场景4: 收入规划

步骤：

1. 基于内容策略估算流量

2. 设定转化率假设

3. 规划收入渠道（课程/咨询/产品）

4. 建立低/中/高收入模型

产出：收入预测 + 优先级决策

Reddit x n8n 集成（详细步骤）

步骤1: 创建 Reddit 应用

1. 登录 Reddit，访问 reddit.com/prefs/apps

2. 滚动到 "Developed Applications" 点击 Create App

3. 填写: Name（如 n8n-test）、App type: Web App、Redirect URI: 你的n8n回调URL

4. 记录 Client ID 和 Client Secret

步骤2: 在 n8n 配置 OAuth2

1. n8n 中创建 OAuth2 API 凭证

2. Authorization URL: https://www.reddit.com/api/v1/authorize

3. Access Token URL: https://www.reddit.com/api/v1/access_token

4. Scope: read identity history

5. Auth URI Query: duration=permanent

用途

监控子版块关键词、跟踪竞品提及、发现内容创意、识别客户痛点

Apify Google Maps 本地潜客生成

高效技巧

先在 Apify 运行抓取，然后用 n8n 的 HTTP Get 节点获取数据。这样可以避免 API 限制、超时和不必要的费用！

配置步骤

Step 1: 选择 Google Maps Scraper

推荐: compass/crawler-google-places

成本: ~$4/1000个商家

Step 2: 配置搜索参数

搜索词: "outdoor kitchen designer"

位置: "Houston, Texas"
